# Cattery Management

Last reviewed: 2026-06-12

This file uses the requested filename `kennel-management.md`, but CatStays product language should be cattery management. CatStays is for cats and catteries only.

## Current Management Areas

CatStays currently includes routes and concepts for:

- Rooms and capacity.
- Bookings.
- Customers.
- Cat profiles.
- Calendar.
- Room planner.
- Accounting.
- Messages.
- Promotions.
- Social media.
- Cat photo updates.
- Payment setup.
- Website editor.
- Marketing.
- Subscription management.

## Target Cattery Operations

The dashboard should help cattery owners:

- See arrivals and departures.
- Accept or decline booking requests.
- Check availability.
- Allocate rooms.
- Manage cat profiles and special care notes.
- Collect deposits and payments.
- Communicate with owners.
- Send photo updates.
- Track revenue and occupancy.
- Run the cattery from a phone.

## Current Risks

- Some dashboard routes still use demo data.
- Some planner data uses localStorage.
- Desktop dashboard layout needs richer use of available screen space.
- Role permissions for owner, staff, customer, and CatStays admin need tightening.
